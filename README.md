# Smart Lens for Visual Assistance

## Overview

Smart Lens for Visual Assistance is a computer vision application designed to assist visually impaired users by detecting and reading text from images.

## Technologies Used

* Python
* OpenCV
* PyTesseract

## Features

* Text detection from images
* Optical Character Recognition (OCR)
* Text-to-Speech conversion
* Real-time visual assistance
